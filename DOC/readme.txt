1、工程目的

	实现温湿度的测量和显示，并持续更新显示

2、工程所用到的器材

	硬件：DHT11温湿度传感器，3.2寸ILI9341LCD电阻屏，秉火stm32f103MINI开发板，串口下载线
	软件：Keil uVsion5、串口调试助手、串口下载工具 
	
3、工程实现模块
	
	串口通信模块：	USART配置，实现串口的接收功能，使用串口调试助手，实现PC机与MDK的交互，显示温湿度的数据
	时钟模块:		系统滴答定时器，用作准确延时
	显示模块：		3.2寸ILI9341LCD电阻屏，用FSMC驱动，在LCD上显示温湿度的数据
	测量模块：      DHT11温湿度传感器将获取的数据发送给MDK处理
	
4、涉及到的知识

	串口通信、系统滴答定时器、GPIO口的配置、DHT11温湿度传感器的工作时序、FSMC的配置和原理、LCD显示原理
	
5、开发方法

	库开发

6、参考文件
	STM32_参考手册、ILI9341参考手册、STM32固件库使用手册、零死角玩转STM32
